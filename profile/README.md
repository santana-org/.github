## 🙌 Contributing

Contributions are welcome across all **santana** projects.
Each repository defines its own workflow, open tasks, and contribution guidelines.

To get started, check the Issues tab and look for labels such as `good first issue`
or `help wanted` to find tasks suitable for new contributors.

## 🔐 Security

If you discover a security vulnerability, please do not open a public issue.
Instead, follow the responsible disclosure process described in
[SECURITY.md](SECURITY.md)

[Sponsor us on GitHub](https://github.com/sponsors/santana-org) to support the
organization and help sustain ongoing development.
